# Advanced React — Persistence Section

Part of [Scrimba's Fullstack Developer Path](https://v2.scrimba.com/the-frontend-developer-career-path-c0j), covering the persistence section of the Advanced React module.

## Stack

- React 19, Vite 7
- Supabase (persistence)
- oxlint + oxfmt (linting & formatting)

## Scripts

| Command             | Description       |
| ------------------- | ----------------- |
| `npm run dev`       | Start dev server  |
| `npm run build`     | Production build  |
| `npm run lint`      | Lint with oxlint  |
| `npm run lint:fix`  | Lint and auto-fix |
| `npm run fmt`       | Format with oxfmt |
| `npm run fmt:check` | Check formatting  |
